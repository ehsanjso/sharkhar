2. **REST API** - Modern API at `http://pi.hole/api/` for programmatic access
3. **Regex filtering** - Block thousands of domains with single patterns
4. **Community regex lists** - mmotti/pihole-regex adds powerful pattern-based blocking
5. **Local DNS records** - Create custom `.local` domains for services (mission-control.local, etc.)
6. **DHCP server mode** - Pi-hole can replace router DHCP for automatic DNS assignment
7. **Integration opportunities** - Can integrate with Mission Control, Uptime Kuma, Home Assistant
