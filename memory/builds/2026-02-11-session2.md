### weekly-report.sh ✅

Created a wrapper script for generating formatted weekly summaries using devlog.sh --range under the hood.

**Features:**
- `--last` - Quick shortcut for last week
- `--offset N` - Go back N weeks
- `--search TERM` - Filter entries by keyword  
- `--summary` - Stats only, skip full entries
- `--markdown` - Export-friendly format
