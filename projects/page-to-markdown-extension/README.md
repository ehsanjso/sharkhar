## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select this folder

## Usage

1. Navigate to any webpage
2. Click the floating purple button (bottom right)
3. The page content is saved as a `.md` file

## Output

The markdown file includes:
- Page title as H1
- Source URL
- Save date
- All text content, headings, links, code blocks, etc.

## Tips

- Drag the button to reposition it
- Works best on article/content pages
- Files are saved to your default Downloads folder
